# Mohammad Alfarras

Personal brand portfolio built with Next.js App Router, Tailwind CSS, and Framer Motion.

## Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- Framer Motion
- Supabase
- Playwright

## Local development

```bash
npm install
npm run dev
```

## Quality checks

```bash
npm run lint
npm run typecheck
npm run build
npm run test:e2e -- tests/e2e/theme-rebuild.spec.ts
```

## Deployment

The project is linked to Vercel and is intended to deploy automatically from `main`.
